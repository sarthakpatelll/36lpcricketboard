# Gallery Feature Added ✅

## Admin:
1. Login to admin panel → Gallery (new dashboard card)
2. Add images: Select date, upload JPG/PNG (max 5MB), add description, Active
3. Toggle/Delete as needed (deletes file too)

## Public:
1. Homepage nav → 🖼️ Gallery
2. Date-wise albums, click images for fullscreen lightbox
3. Responsive grid, lazy loading

## Files Created/Updated:
- DB: `gallery_images` table
- `admin/gallery.php`: CRUD
- `gallery.php`: View albums
- Nav: + Gallery link
- Admin dashboard: + Gallery card
- CSS: Gallery styles (hover, responsive)
